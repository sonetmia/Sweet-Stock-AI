# Sweet-Stock-AI
# 🍬 Sweet AI — Microstock Assistant Chatbot

Sweet AI হলো একটি বুদ্ধিমান চ্যাটবট, যা OpenRouter API ব্যবহার করে চলে।  
এটি **Microstock Journey With Sonet** কোর্সের সম্পূর্ণ তথ্য স্থায়ীভাবে জানে এবং যেকোনো প্রশ্নের উত্তর দিতে পারে।

এছাড়া ইউজার `/remember` কমান্ড দিয়ে নিজস্ব মেমোরি যোগ করতে পারে, যা ব্রাউজারে সংরক্ষিত থাকে।

## ✨ ফিচারসমূহ

- 🤖 **OpenRouter API** ইন্টিগ্রেশন (Gemini Flash Lite ফ্রি মডেল)
- 📚 **স্থায়ী কোর্স জ্ঞান** — পুরো Microstock Journey কোর্সের তথ্য
- 🧠 **কাস্টম মেমোরি** — `/remember`, `/forget`, `/memories` কমান্ড
- 💾 **localStorage**-এ ব্যক্তিগত মেমোরি সংরক্ষণ
- 🎨 **সুন্দর ও রেসপনসিভ UI** (গ্রেডিয়েন্ট থিম, চ্যাট বাবল)
- 🔒 **API key নিরাপদ** — সার্ভার সাইড API route-এর মাধ্যমে
- 🚀 **Vercel-এ সহজ ডেপ্লয়মেন্ট**

## 🧰 প্রযুক্তি স্ট্যাক

- [Next.js 14](https://nextjs.org/) (App Router)
- React 18
- OpenRouter API
- CSS-in-JS (inline styles)
